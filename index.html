AquaSentry: Smart Water Sentinel

Integrated Solar-Powered Water Quality Monitoring and **High-Fidelity Local Data Logging** System.

1. Sensor Array and Processing Sub-Systems

Aquatic Probes & Calibration Array

Data Acquisition: **TDS (0–1000 ppm)**, **pH (0–14, $\pm 0.1$)**, **Turbidity (0–1000 NTU)**, and **DS18B20 Temperature** sensor. Fixed inside a waterproof sensor node.

Multi-parameter input for high-fidelity water analysis.

Control Unit & Local Display

Processing Core: **ESP32 Microcontroller** (Dual-core, low power sleep). Executes all data acquisition, complex calculations, and direct local storage to the MicroSD card. Features a **1.3″ OLED Display** for color-coded status (Safe, Moderate, Unsafe).

Central logic processing and powerful data management interface.

SD Card Interface & Local Access Port

Connectivity: Includes a **MicroSD Card Module** for robust local data logging. The unit provides a sealed **UART/Serial port** for physical data retrieval and device diagnostics only. **No external network connectivity (Wi-Fi/GSM) is enabled or utilized.**

Ensures secure, non-volatile data storage and physical access for management.

Autonomous Power & Data Log

Sustainability: **5V, 2W Solar Panel** and **3.7V, 2500mAh Li-ion Battery**. Managed by **TP4056** and **Voltage Regulator (LM2596)**. Local backup via **MicroSD Card**.

Sustainable field operation with non-volatile local storage.

2. Mechanical Protocol & Physical Integration

Fluid Dynamic Interface

The sensor node is attached directly below the tap or handpump. A secure mounting system ensures the probes remain immersed during sampling. The main control box, sealed with an **IP-rated poly-casing**, is fixed nearby.

  • Deployment Vector: Control box is mounted laterally to the pump structure. Display orientation must be optimized for community accessibility and viewing distance.
  • Energy Harvester: **5V Solar panel** alignment requires maximum daily irradiance, typically mounted at a **45° angle** above the control box.
  • Maintenance Protocol: The waterproof **probe module is detachable** for scheduled cleaning/calibration cycles, ensuring long-term data fidelity.

3. Component Breakdown & Role Analysis (BOM)

ESP32 Microcontroller

Central Brain & Local Logic Unit

Role: Executes the core loop: data acquisition, complex sensor calibration, display updates, and **storage to the MicroSD card**. Its integrated Wi-Fi is disabled for this offline application to conserve maximum power.
Spec: Dual-core MCU, used for high-efficiency local data handling.

Sensor Array (TDS, pH, Turbidity)

Water Quality Metrics

Role: Core instruments for measuring water safety. They provide the raw data required by the ESP32 to calculate the final safe/unsafe verdict.
Spec: TDS ($0 - 1000 \text{ ppm}$), pH ($0 - 14$), Turbidity ($0 - 1000 \text{ NTU}$).

Solar & Battery System

Autonomous Energy Source

Role: Provides reliable power without external electricity. The solar panel charges the Li-ion battery via the TP4056 charge controller, supplying stable voltage via the LM2596 regulator.
Spec: $5 \text{V}, 2 \text{W}$ Solar, $3.7 \text{V}, 2500 \text{mAh}$ Li-ion.

MicroSD & Display Unit

Local Feedback & Backup

Role: The display shows real-time data and the color-coded status for immediate community access. The MicroSD provides non-volatile local backup for all historical readings, which must be retrieved manually.
Spec: $1.3''$ OLED/TFT display. MicroSD logging module.
